Проект интерпретатора языка программирования с 4 типами данных : int,double,bool,string
и несколькими управляющими конструкицями if else while.
из операторов сравнение пока есть только < для int, но в будущем можно добавить остальные операторы
есть также два слова для упрвление вводом и выводом:
PRINT - вычисляет и выводит следущее выражение,
INPUT - записывает введенное значение в _input, тип _input выводится сам:
int - задаётся целым числом,
double - задается числом с точкой,
bool - словами true или false,
string с помощью кавычек "CONTENT".
